# AI Companion Bot

Folder repository ini dikhususkan untuk bagian backend service. Backend menggunakan FastAPI sebagai framework yang menyediakan utilitas untuk aplikasi AI Companion

## Menjalankan Project

**Install Library**

```bash
pip install -r requirements.txt
```

**Menjalankan Services**

```bash
uvicorn app.main:app --reload
```

**Akses URL**
http://127.0.0.1:8000
